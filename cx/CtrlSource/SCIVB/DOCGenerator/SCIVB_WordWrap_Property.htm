PropertyName WordWrap [A WrapStyle Value. The Property is Read-Write] - If set to true the document will wrap lines which are longer than itself. If false then it will dsiplay normally.

If set to true the document will wrap lines which are longer than itself. If false then it will dsiplay normally.

Type

A WrapStyle Value. The Property is Read-Write

Syntax

Object.WordWrap [ = WrapStyle ]

The WordWrap Property syntax has these parts:

Part
object An expression evaluating to an Object of type SCIVB.
WrapStyleA WrapStyle Value. The Property is Read-Write.
